Portfolio Web (Trabajo final para Argentina Programa)

En este proyecto se me encomendó realizar una aplicación web referida a mi portfolio donde se puso en práctica la arquitectura Cliente-Servidor, para poder realizarla se utilizaron los leguajes html, css y java como lenguajes bases de Programación, utilice Angular como framework para desarrollar la aplicación web desarrollada en TypeScript implementando los lenguajes html y css para el desarrollo de la vista y estilo (Frontend) y a su vez también utilice NetBeans que es un entorno de desarrollo integrado libre, hecho principalmente para el lenguaje de programación Java, en el cual realice la base de datos con la intervencion del programa MySQL Workbench (Backend). Además, Utilizando otras herramientas para todo tipo de pruebas (Xampp, Postman).
